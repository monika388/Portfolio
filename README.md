# Monika • React portfolio

A calming, expressive, and professional portfolio built with React. It showcases projects, skills, education (SPU, BCA), and TechElecon internship experience, with responsive layouts, gradients, custom fonts, and modular components.

## Tech stack
- React • HTML • CSS • JavaScript
- Modular components: Home, About, Skills, Education, Projects, Contact

## Features
- Responsive layout using Flexbox/Grid
- Modular architecture with clean file structure
- Calm color palette, rounded corners, gradients, custom fonts
- Resume embedded/downloadable (PDF)
- Mobile-first spacing and typography
- Project cards with live/demo links
